jQuery.fn.leko

获取某HTML Element对应于某插件的隐含对象。该隐含对象使用jQuery的data("plugin-name")方法创建，通过读取并解析该Element在HTML代码的Tag声明中使用JSON语法设置的对应于插件名称的自定义属性值，为插件的初始化提供该对象的私有默认设置(优先权高于插件的全局默认设置和插件的运行时参数设置)。此外，该隐含对象还用于插件在针对该Element的工作中存取私有变量。

语法

object = jQuery(selector).leko(plugin-name)

参数

参数名是否可选缺省值数据类型可能的取值说明
plugin-namestring插件名称。

返回

数据类型可能的取值说明
object对获取到的对象的引用。

示例

<a colorShade="{color:'red'}">Test</a>

<script>

$("a").leko("colorShade").color="blue";
$("a").colorShade();

</script>
