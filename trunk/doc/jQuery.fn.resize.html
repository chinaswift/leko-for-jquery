jQuery.fn.resize

基于网页元素、文档、窗口的尺寸为基础调整当前元素的尺寸。

语法

jQuery = jQuery(selector).resize([target] , [w] , [h] , [offset-w] , [offset-h] , [rect])

参数

参数名是否可选缺省值数据类型可能的取值说明
target√windowstring合法的jQuery选择符。将被转换为jQuery对象。
element已存在的DOM元素。将被转换为jQuery对象。
jQuerywindow基于视口尺寸。
document基于文档全内容区域尺寸。
others基于所包含网页元素的位置和布局尺寸。
w√2number0以指定元素的 style.width 为目标宽度。
1以指定元素的 innerWidth 为目标宽度。当参数 target 为 window 或 document 时无效。
2以指定元素的 outerWidth 为目标宽度。当参数 target 为 window 或 document 时无效。
others维持原有宽度。
h√2number0以指定元素的 style.height 为目标高度。
1以指定元素的 innerHeight 为目标高度。当参数 target 为 window 或 document 时无效。
2以指定元素的 outerHeight 为目标高度。当参数 target 为 window 或 document 时无效。
others维持原有高度。
offset-w√0number以像素为单位的宽度调整值。
offset-h√0number以像素为单位的高度调整值。
rect√[ -Number.MAX_VALUE, Number.MAX_VALUE, Number.MAX_VALUE, -Number.MAX_VALUE ]array如[min-height, max-width, max-height, min-width]格式的rect数组，用于限定可改变大小范围。不需要限定的项，用null替换即可。

返回

数据类型可能的取值说明
jQuerythis

示例

$("#box1").resize(document);

$("#box1").resize($("#box2"),0,2);
