jQuery.fn.move

基于网页元素、文档、窗口、鼠标等的布局或位置定位当前元素。

语法

jQuery = jQuery(selector).move([target] , [x] , [y] , [offset-x] , [offset-y] , [rect])

参数

参数名是否可选缺省值数据类型可能的取值说明
target√windowstring合法的jQuery选择符。将被转换为jQuery对象。
element已存在的DOM元素。将被转换为jQuery对象。
jQuerywindow基于视口定位。
document基于文档全内容区域定位。
event返回触发事件的鼠标位置定位。仅限鼠标类事件。
others基于所包含网页元素的位置和布局定位。
x√1number0在指定元素的左外侧。当参数 target 为 window 或 document 时无效。
1在指定元素的左内侧。
2在指定元素的横居中。
3在指定元素的右内侧。
4在指定元素的右外侧。当参数 target 为 window 或 document 时无效。
others维持原有横坐标。
y√4number0在指定元素的上外侧。当参数 target 为 window 或 document 时无效。
1在指定元素的上内侧。
2在指定元素的竖居中。
3在指定元素的下内侧。
4在指定元素的下外侧。当参数 target 为 window 或 document 时无效。
others维持原有纵坐标。
offset-x√0number以像素为单位的水平位置调整值。仅当参数 x 指定了合法数字枚举值时才有效。
offset-y√0number以像素为单位的垂直位置调整值。仅当参数 y 指定了合法数字枚举值时才有效。
rect√[ -Number.MAX_VALUE, Number.MAX_VALUE, Number.MAX_VALUE, -Number.MAX_VALUE ]array如[top, right, bottom, left]格式的rect数组，用于限定可移动边界范围，基于文档全内容区域定义。超出范围的将自动被限制到边界内。

返回

数据类型可能的取值说明
jQuerythis

示例

$("#popup").move($("#block"),1,4);

$("#popup").move(window,2,2,10,0);

$(document).click(function(e){
    $("#popup").move(e,0,0);
});
