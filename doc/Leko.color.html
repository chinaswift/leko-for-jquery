Leko.color

将CSS3语法的色彩定义值或RGBA数组转换为当前用户浏览器适用的色彩定义值。

语法

string|undefined = Leko.color(color)

参数

参数名是否可选缺省值数据类型可能的取值说明
colorstring各种符合CSS3语法格式的颜色定义。包括：

  1. rgb (R,G,B)
  2. rgba(R,G,B,A)
  3. hsl (H,S,L)
  4. hsla(H,S,L,A)
  5. #RGB
  6. #RRGGBB
  7. ...

详见实例。
array包含RGBA四个通道数值的数组。

返回

数据类型可能的取值说明
string当前用户浏览器适用的色彩定义值。视当前浏览器的兼容性，可能是rgba()也可能是rgb()定义。
undefined转换失败。

示例

var x=_.color([255,100,50,0.5]);
alert(x);



var x=_.color("rgba(255,100,50,0.5)");
alert(x);
